# Formato de preguntas para el simulador

Cada pregunta debe estar en una sola línea, con el siguiente formato:

Pregunta|Opción A|Opción B|Opción C|Opción D|Letra de la respuesta correcta

Ejemplo:
¿Cuál es el color del semáforo que indica detenerse?|Rojo|Verde|Amarillo|Azul|A

- Si una pregunta tiene menos de 4 opciones, dejar los campos vacíos.
- La letra de la respuesta correcta debe ser A, B, C o D (en mayúscula).
- No incluir numeración ni encabezados.

Este formato es necesario para que el simulador funcione correctamente.
